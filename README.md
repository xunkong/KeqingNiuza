# 刻记牛杂店 最终版

本项目已停止维护，最后发布的最终版是一个独立的、没有后续更新的版本。

## 功能介绍

- 祈愿记录加载及分析
- 每日/每周任务提醒
- 自动演奏风物之诗琴
